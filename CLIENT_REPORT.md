### 核心价值
✅ **教学工具** - 可视化展示强化学习算法

# 🎯 项目成果

## 二、功能特性

### 1. 用户系统 👥
- ✅ 完整的注册/登录机制
- ✅ 数据严格隔离（每个用户独立数据）
- ✅ 安全的密码管理（pbkdf2:sha256 加密）
- ✅ Session 会话管理

### 2. 模型训练 🤖
- ✅ **20+ 个强化学习模型**支持：
  - Attention Model (AM)
  - POMO
  - SymNCO
  - MatNet
  - MDAM
  - DeepACO
  - 等等...

- ✅ **6 种优化问题**支持：
  - TSP (旅行商问题)
  - CVRP (车辆路径问题)
  - OP (定向问题)
  - PCTSP (带奖励旅行商)
  - PDP (取送货问题)
  - SDVRP 等

### 3. 实时监控 📊
- ✅ 训练进度条（实时更新）
- ✅ Loss/Reward 动态曲线（SSE 推送）
- ✅ 训练日志实时显示
- ✅ 最佳性能指标追踪

### 4. 结果可视化 📸

### 5. 文件管理 📁

- ✅ 一键下载功能

### 6. 数据分析 📈
- ✅ **算法对比页面** - KDD 2025 论文 benchmark
- ✅ **性能数据** - 20/50 节点问题对比
- ✅ **交互式图表** - ECharts 数据可视化

### 7. 知识库 📚

# 🏗️ 技术架构

## 三、系统架构

```
┌─────────────────────────────────────────────────┐
│           用户浏览器 (Web 前端)                   │
│  HTML5 + CSS3 + JavaScript (ES6+)                │
│  实时数据推送 (Server-Sent Events)              │
└────────────────┬────────────────────────────────┘
                 │
                 ↓ HTTPS/HTTP
┌─────────────────────────────────────────────────┐
│         Nginx 反向代理 (负载均衡)                │
│  - SSL/TLS 加密
│  - 静态文件缓存
│  - 反向代理
└────────────────┬────────────────────────────────┘
                 │
                 ↓
┌─────────────────────────────────────────────────┐
│      Gunicorn WSGI 应用服务器 (4+ Workers)      │
│  - Flask Web 框架
│  - 多进程并发处理
│  - 会话管理
└────┬────────────────────┬──────────────────────┘
     │                    │
     ↓                    ↓
┌──────────────┐   ┌──────────────────┐
│  MySQL 数据库 │   │  PyTorch GPU 训练 │
│  用户数据     │   │  RL4CO 库        │
│  训练结果     │   │  Lightning       │
└──────────────┘   └──────────────────┘
     │                    │
     ├────────┬───────────┤
     │        │           │
     ↓        ↓           ↓
  用户数据  模型检查点   训练文件
  训练会话  可视化结果   GIF动画
```

# 💼 部署方案

## 八、三种部署方案

### 方案 1：Systemd 服务（生产推荐）

```
优点: 稳定可靠、性能最好、支持开机自启
缺点: 需要 Linux 知识、手动配置较多

部署时间: 45 分钟
年度成本: 1800-2400 RMB (阿里云 2核4GB)
```

**架构**:
```
Nginx (HTTP/HTTPS) 
  ↓
Gunicorn (多进程)
  ↓
Flask 应用 ← MySQL 数据库
```

### 方案 2：Docker Compose（开发推荐）

```
优点: 快速部署、一键启动、环境隔离
缺点: 资源占用稍多、需要学习 Docker

部署时间: 20 分钟
年度成本: $120-150 (DigitalOcean $10/月)
```

**优势**: 数据库、应用、Nginx 全部包含，开箱即用

### 方案 3：Kubernetes（大规模推荐）

```
优点: 自动扩展、负载均衡、全球部署、99.99% 可用性
缺点: 复杂度高、运维成本高

部署时间: 1-2 天
年度成本: 中等以上 (AWS、Google Cloud)
```
