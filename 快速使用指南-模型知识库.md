# 快速使用指南 - 模型知识库功能

## 🎯 功能简介

在首页新增了"模型知识库"模块，用户可以查询20多种强化学习模型的详细信息，包括论文出处、核心原理、技术架构、性能数据等。

## 📍 位置

首页 → 配置区域 → **模型知识库**（问题类型选择区域下方）

## 🚀 使用步骤

### 1. 选择模型
在"选择模型"下拉菜单中选择您感兴趣的模型。模型按类别分组：
- 构造方法（自回归）
- 构造方法（非自回归）
- 改进方法
- 强化学习算法
- 传导式强化学习

### 2. 查看详情
点击 **🔍 查看详情** 按钮，在新窗口中打开模型详情页面。

### 3. 浏览信息
模型详情页面包含：
- **核心定位**：模型的核心思想
- **技术架构**：详细的架构说明
- **关键创新点**：突出的创新特性
- **性能表现**：实验数据和性能指标
- **优势与局限**：全面的评价
- **适用场景**：推荐使用场景
- **论文链接**：可直接跳转到原始论文

### 4. 返回首页
点击页面底部的 **← 返回首页** 按钮。

## 📚 推荐模型

### 新手入门
- **AM (Attention Model)**：最经典的模型，易于理解
- **PtrNet (Pointer Network)**：开创性工作，概念简洁
- **REINFORCE**：最基础的强化学习算法

### 追求性能
- **POMO**：性能优异，速度快（推荐！）
- **Sym-NCO**：当前SOTA之一，质量最优
- **MatNet**：表达能力强，大规模问题优势明显

### 创新研究
- **DeepACO**：结合传统优化与深度学习
- **Active Search / EAS**：测试时优化，质量提升显著
- **MTPOMO**：多任务学习，知识迁移

### 训练算法
- **REINFORCE**：最简单，易于实现
- **A2C**：更稳定，方差更小
- **PPO**：工业界首选，最可靠

## 💡 使用技巧

1. **对比学习**：同时打开多个模型页面，对比它们的优劣势
2. **由浅入深**：先了解经典模型（AM, PtrNet），再学习现代方法（POMO, Sym-NCO）
3. **论文阅读**：对感兴趣的模型，点击论文链接深入学习
4. **实践验证**：在训练页面选择不同模型进行实验，验证其性能

## 📊 模型选择建议

| 场景 | 推荐模型 | 理由 |
|------|---------|------|
| 快速原型 | AM | 实现简单，速度快 |
| 生产部署 | POMO | 性能与速度平衡最好 |
| 学术研究 | Sym-NCO | SOTA性能，理论扎实 |
| 大规模问题 | MatNet / DeepACO | 专为大规模设计 |
| 时间充裕 | EAS + POMO | 质量最优 |

## ⚠️ 注意事项

1. 部分模型标注为"敬请期待"，表示功能正在开发中
2. 论文链接若显示为"#"，表示暂无公开论文
3. 性能数据仅供参考，实际效果可能因数据集而异
4. 建议结合实际需求选择模型
