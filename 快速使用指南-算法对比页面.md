## 🔍 常见问题

### Q1: 为什么有些算法在某些问题上没有数据？
**A**: 论文中某些算法没有在特定问题上测试，如Gurobi只测试了TSP。

### Q2: Gap是什么意思？
**A**: Gap = (算法结果 - 最优结果) / 最优结果 × 100%，表示与已知最优解的差距百分比。

### Q3: OP问题为什么Cost越高越好？
**A**: OP（定向问题）目标是最大化收集的奖励，所以Prize值越高越好，但Gap仍然是越小越好。

### Q4: 时间单位不统一怎么比较？
**A**: 选择"Time"指标时，系统会自动将所有时间转换为秒进行比较。

### Q5: 如何导出图表？
**A**: 右键点击图表 → 选择"保存为图片"。

## 🚀 进阶技巧

### 1. 快速对比多个问题
```
步骤：
1. 固定其他维度（如50节点、Gap、Greedy）
2. 依次切换问题类型（TSP→CVRP→OP...）
3. 观察不同问题的算法表现差异
```

### 2. 评估算法一致性
```
步骤：
1. 选择一个算法（如POMO）
2. 切换不同问题类型
3. 查看该算法在不同问题上的排名

示例: POMO在所有问题上Gap排名前3
```

### 3. 寻找性价比最高的算法
```
步骤：
1. 筛选条件: 50节点 + TSP + Gap + 全部算法
2. 找到Gap<1%且Time<1m的算法
3. 结果: POMO (Gap=0.89%, Time=<1s)
```
